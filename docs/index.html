Sveetoy

A Sass library for programmatic rythm and some candies on top of Foundation for Sites (currently for Foundation 5.5.3).

Responsive rhythm for your spaces and texts

Enforce your color policy from settings

Faster prototyping

Download version 0.5.1 (ZIP)

We recommend you to use it as an imported library from your Sass compiler and avoid to include it in your Sass source files.

Once library is available with your compiler, import it in your sources:

@import "settings";

@import "foundation";

@import "sveetoy";

You may notice that Foundation components and settings must be loaded before Sveetoy (because some components use some of its settings).

In settings source, you can define Sveetoy settings you want to override.

Typography

This component adds some minor things to the Foundation typography stuff.

Also there is a new rule that prevent paragraphs to have a bottom margin if they are the last child of their parent.

Can be disabled with
$include-sveetoy-typography: false;

Spaces

This component add styles to apply paddings using space rythmic defined in map $sveetoy-spaces.

Padding can be applied for both sides on vertical or horizontal axes or optionally on only one side.

There are many space sizes available and each of them have their own breakpoint sizes.

Can be disabled with
$include-sveetoy-spaces: false;

Sections

A section is a full width container useful for fluid design layouts but with first goal to set background and font colors.

Available features can apply various vertical padding using rythmic space from $sveetoy-spaces and various color schemes from $sveetoy-colors-schemes.

.section does not have any links with <section> element although you can mix them if you want.

Can be disabled with
$include-sveetoy-sections: false;

Flexbox

Foundation for site version 5 does not have any flexbox grid and version 6 force to choose between float and flexbox grid.

So here it is, a simple flexbox grid.

Alike .row in float grid, .flex-grid is the main container and .cell is equivalent to .columns.

Available cell sizes are builded from $flex-grid-sizes list and defined for each Foundation media query.

.margin feature is defined from $flex-grid-gutter.

Available cell width sizes are builded from list $flex-grid-sizes, note that size decimals are removed from the size class name, so value 33.333 will build a class name like .small-33, .medium-33, etc..

Can be disabled with
$include-sveetoy-flex: false;
Source location
sveetoy/components/_flex.scss
