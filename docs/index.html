Sections

Section is just a full width container on top level of your body elements tree. There are useful for fluid design layouts but their first goal is to set background and font colors for their child elements.

Default section have vertical padding but you can use bigger or lower padding value or even null.

Section CSS rules enforce color for texts. You can set different font color for paragraph (lists inherit from it), each title level and links.

.section does not have any links with <section> element although you can mix them if you want.

Can be disabled with
$include-sveetoy-sections: false;
Source location
sveetoy/components/_sections.scss

Flexbox

Foundation for site version 5 does not have any flexbox grid and version 6 force to choose between float and flexbox grid.

So here it is, a simple flexbox grid.

Alike .row in float grid, .flex-grid is the main container and .cell is equivalent to .columns.

Available cell sizes are builded from $flex-grid-sizes list and defined for each Foundation media query.

.margin feature is defined through $flex-grid-gutter.

Can be disabled with
$include-sveetoy-flex: false;
Source location
sveetoy/components/_flex.scss
