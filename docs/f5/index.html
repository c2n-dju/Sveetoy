Sveetoy

A Sass library for programmatic rythm and some candies on top of Foundation for Sites version 5.5.3 or 6.3.1.

Responsive rhythm for your spaces and texts

Enforce your color policy from settings

Faster prototyping

Install

We recommend you to use it as an imported library from your Sass compiler and avoid to include it inside your project Sass sources.

Once library is available with your compiler, import it in your sources:

@import "settings";

@import "foundation";

@import "sveetoy";

Alike with Foundation, you must load the settings first then libraries. In your settings you can define both Foundation and Sveetoy settings or you may split them in two or more settings files.

Included file sveetoy/_f5_settings.scss contains some sample settings you may include in at end of your main settings file and adjust to your needs. Be aware that some of its values are linked to some Foundation setting values.

Since Sveetoy 0.6.0, default enabled support is for Foundation 6, for Foundation 5 support you will need to activate it in your settings with following variable to place before Sveetoy import:

$sveetoy_foundation_support: 5;

Flexbox

Foundation for site version 5 does not have any flexbox grid and version 6 force to choose between float and flexbox grid.

So here it is, a simple flexbox grid.

Alike .row in float grid, .flex-grid is the main container and .cell is equivalent to .columns.

Available cell sizes are builded from $flex-grid-sizes list and defined for each Foundation media query.

.margin feature is defined from $flex-grid-gutter.

Available cell width sizes are builded from list $flex-grid-sizes, note that size decimals are removed from the size class name, so value 33.333 will build a class name like .small-33, .medium-33, etc..

Can be disabled with
$include-sveetoy-flex: false;
Source location
sveetoy/components/_flex.scss
