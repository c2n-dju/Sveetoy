Buttons

This component provides some features for buttons.

Also there is a new rule that prevent buttons to have a bottom margin if they are the last child of a div, p, dd, li element.

Can be disabled with
$include-sv-buttons: false;
Source location
sveetoy/components/_buttons.scss

Important

Make the button text stronger and uppercase.

{% for name in ['', 'important'] %}
.button{% if name %}.{{ name }}{% endif %}
{% endfor %}

Sizes

Add .full feature that act like .expand but preserve normal horizontal padding.

.button.full

Paddings

Add .modest feature so the button padding are divided by two or less, it depend from size definitions in $sv-button-modests.

{% for name in ['', 'small.modest', 'modest', 'large.modest', 'small.modest.radius', 'modest.round'] %}
.button{% if name %}.{{ name }}{% endif %}
{% endfor %}

Colors

Button color features are builded from $sv-button-colors-schemes that is on the same format than $sv-colors-schemes except additional (and optionnal) border-color option.

{% endfor %}
