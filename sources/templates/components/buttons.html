Buttons

Buttons are a common way to display important links, often called CTA (*Call To Action*).

They define a background, padding and font to be visually important.

This component only add .modest feature every other showed features are from Foundation and just for comparison.

Can be disabled with
$include-sveetoy-buttons: false;
