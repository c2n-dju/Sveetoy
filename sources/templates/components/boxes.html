Boxes

Box component is a combination of spaces and colors settings, alike .block but with both horizontal and vertical padding and optional borders.

Can be disabled with
$include-sveetoy-boxes: false;
Source location
sveetoy/components/_boxes.scss

Spaces

Space features are built from $sveetoy-spaces (except for .unpadded that just set padding to 0). Default box colors comes from normal item from $sveetoy-spaces.

Note than first direct child element does not have a margin top and last direct child element does not have a margin bottom.
