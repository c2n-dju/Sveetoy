Blocks

A block is a full width container useful for fluid design layouts but with first goal to set background and font colors.

Available features can apply various vertical padding using rythmic space from $sveetoy-spaces and various color schemes from $sveetoy-colors-schemes.

See Box component for available spacing features.

Can be disabled with
$include-sveetoy-blocks: false;
