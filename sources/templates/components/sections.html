Sections

A section is a full width container useful for fluid design layouts but with first goal to set background and font colors.

Available features can apply various vertical padding using rythmic space from $sveetoy-spaces and various color schemes from $sveetoy-colors-schemes.

.section does not have any links with <section> element although you can mix them if you want.

Can be disabled with
$include-sveetoy-sections: false;
