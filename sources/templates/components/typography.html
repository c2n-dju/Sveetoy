Typography

This component adds some minor things to the Foundation typography stuff.

Also there is a new rule that prevent paragraphs to have a bottom margin if they are the last child of their parent.

Can be disabled with
$include-sveetoy-typography: false;
Source location
sveetoy/components/_type.scss

Title element

Title styles are not anymore sticked to hX elements, now you can use them by the way of .title-X classes on any other element to preserve semantic. You may even use a .title-X class on hX element.

{% for count in range(1, 7) %} h{{ count }}, .title-{{ count }} {% endfor %}

Text colors

Text colors are available for every color defined in map $sveetoy-color-text-palette. Properties are defined using !important flag so take care of their usage.

{% for name in ['white', 'black', 'gray25', 'gray82', 'gray88', 'lightskyblue', 'deepskyblue4'] %}

.text-{{ name }}

{% endfor %}

Thickness

Two more thickness are available if their corresponding variable is defined. $font-weight-thin for thinest thickness and $font-weight-strong for boldest thickness.

{% for name in ['thin', 'normal', 'bold', 'strong'] %}

{% if name == 'bold' %}<b>, {% elif name == 'strong' %}<strong>, {% endif %}.text-{{ name }}

{% endfor %}

Small

Add some size possibilities to <small> element. .medium is equivalent to the default <small> behavior. Available sizes are built from $sveetoy-smalls.

{% for name in ['small', 'medium', 'big'] %}

.{{ name }} {{ lipsum(n=1, html=False, min=15, max=16) }}

{% endfor %}

{{ lipsum(n=1, html=False, min=15, max=16) }}

Transform and decoration

.text-upper enables you to correctly write your texts in lowercase but render them as uppercase. .text-important just define the underline value for text-decoration property.

.text-upper

.text-important

Underline

Various underline positions, thickness and widths. All of them are implemented with border properties on a pseudo-element :after.

Width

Built from list $sveetoy-underline-widths.

{% for name in ['tiny', 'short', 'normal', 'large', 'full'] %}

.underline{% if name %}.{{ name }}{% endif %}

{% endfor %}

Thickness

Built from map $sveetoy-underline-thickness.

{% for name in ['thin', 'normal', 'bold'] %}

.underline{% if name %}.{{ name }}{% endif %}

{% endfor %}

Distance

Built from list $sveetoy-underline-distances.

{% for name in ['sticked', 'near', 'normal', 'far'] %}

.underline{% if name %}.{{ name }}{% endif %}

{% endfor %}

Alignment
