Install

We recommend you to use it as an imported library from your Sass compiler and avoid to include it inside your project Sass sources.

Once library is available with your compiler, import it in your sources:

@import "settings";

@import "foundation";

@import "sveetoy";

Alike with Foundation, you must load the settings first then libraries. In your settings you can define both Foundation and Sveetoy settings or you may split them in two or more settings files.

Included file sveetoy/{% if foundation_version == 5 %}_f5_settings{% else %}_f6_settings{% endif %}.scss contains some sample settings you may include in at end of your main settings file and adjust to your needs. Be aware that some of its values are linked to some Foundation setting values.

{% if foundation_version == 5 %}

Since Sveetoy 0.6.0, default enabled support is for Foundation 6, for Foundation 5 support you will need to activate it in your settings with following variable to place before Sveetoy import:

$sveetoy_foundation_support: 5;

{% endif %}
